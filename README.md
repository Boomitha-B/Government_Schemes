## 📖 About The Project

The **Government Schemes Portal** is an informational platform designed to help citizens easily discover and understand various **government welfare schemes in India**.

Many people are unaware of the benefits provided by the government, including programs related to **education, healthcare, financial assistance, agriculture support, and social welfare**. This project aims to bridge that gap by presenting scheme information in a **clear, accessible, and organized format**.

The website acts as a centralized hub where users can explore different government initiatives and learn how they can benefit from them.

---

## ✨ Key Features

✔ **Centralized Information Hub** – Provides details of multiple government schemes in one place.

✔ **Simple Navigation** – Easy-to-use layout that helps users quickly access information.

✔ **Scheme Categories** – Schemes organized by sectors such as education, agriculture, and social welfare.

✔ **Responsive Design** – Works smoothly on desktops, tablets, and mobile devices.

✔ **Awareness Platform** – Helps citizens learn about available government benefits.
